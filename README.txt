Here's the simplest way to try it out on your pc locally.

1. install python 3 from http://python.org
2. start your favorite terminal (shell, dos prompt, ...)
3. navigate to the folder where this README.txt is located
>>> cd /home/development/euclidean_rhythm
4. start python to have a local web server
>>> python3 -m http.server
5. start your browser (if your preferred browser fails, try chrome/chromium as browser),
   and surf to http://localhost:8000
